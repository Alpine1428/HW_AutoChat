 ##🛡️ HolyWorld AutoReply v2.0

**HolyWorld AutoReply** — это продвинутый ассистент для модераторов сервера HolyWorld (версия 1.20.1 Fabric). Мод полностью автоматизирует процесс проверки игроков на читы, общение в чате и выдачу наказаний.

## ✨ Основные возможности

*   **🖥️ Удобное GUI Меню:** Все настройки переключаются одной кнопкой (`Right Shift`).
*   **🤖 Умный авто-ответчик:** Более 60 сценариев ответов (AnyDesk, RustDesk, тайминг, инструкции).
*   **⚡ Авто-действия:** Автоматический ввод команд для начала и завершения проверки.
*   **🚫 Система Авто-Банов:**
    *   **Оскорбление:** Бан 30 дней.
    *   **Отказ:** Бан 30 дней (при написании слова "Отказ").
    *   **Признание:** Бан 20 дней (при написании слова "Я чит").
*   **👀 Отслеживание статуса:** Мод понимает, когда проверка началась, идет или закончилась.

## 🎮 Управление

| Клавиша / Команда | Действие |
| :--- | :--- |
| **`RIGHT SHIFT`** | **Открыть меню настроек (Вкл/Выкл функций)** |
| `/ai start` | Включить мод |
| `/ai stop` | Выключить мод |
| `/ai status` | Показать текущий статус и режим |
| `/ai clear` | Сбросить состояние (если что-то зависло) |
| `/ai test` | Тестовое сообщение |

## ⚙️ Логика работы (Workflow)

1.  **Подготовка:** Вы пишете `/hm spy <ник>` -> `/hm spyfrz`.
2.  **Проверка:** Мод ждет сообщение `[CHECK]` и начинает отвечать игроку.
3.  **Завершение:** При написании `/hm sban...` или `/hm unfrz` мод отключается от игрока.

## 🚫 Авто-Баны

*   **Оскорбление:** `/hm sban <ник> 30d Insult`
*   **Отказ:** При словах "Отказ/Отказываюсь" -> `/hm sban <ник> 30d Refusal`
*   **Признание:** При словах "Я чит/Признание" -> `/hm sban <ник> 20d Confession`
